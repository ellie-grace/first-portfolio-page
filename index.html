About me.

I grew up in the UK, but after graduating from University with a Masters in Aeronautical Engineering, instead of following a traditional career path, I caught the travel bug. I can currently be found in Malta, enjoying the Mediterranean sunshine, but in the last five years, I have lived/worked in five different countries, and travelled in over 15. When I am not working, I can be found doing yoga, scuba-diving or trying to take half-decent photos.

Web development is a new journey for me, having only seriously pursued it full time in the beginning of 2018. Previously, my experience was limited to theoretical modelling at university, and using website builders including Wordpress. I have since completed The Web Developer Bootcamp, and The Advanced Web Developer Bootcamp on Udemy, countless articles, Codeacademy modules and a lot of Youtube tutorials. I am largely self-taught, so I understand how overwhelming website creation can be to a non-tech person. Creating a website is a process, and I aim to make it as enjoyable as possible.

What I do.

I am a freelancer, primarily focusing on front-end web development for small businesses. I love to create custom websites, with no templates, no unnecessary features or bloated code slowing you site down. On the tech side, I write in the following languages: HTML5, CSS3, Javascript, focusing on using Flexbox and vanilla JS for my creations. I have used JQuery and other JS libraries, Bootstrap and Content Management Systems. I am currently working on REACT apps, and have also created full-stack apps using node.js, MongoDB and Express. I can also provide the following services:

development

Writing the code from scratch, I can create bespoke websites, no limiting templates. I strive to make all my sites responsive, mobile-friendly and browser cross-compatible.

design

Designing a website is more than pretty colours and fonts, I can take your initial ideas and help to create complete design, that is user focused and easy to navigate.

support

From hosting plans, to backups and updates, maintaining a website can be hard work. The internet is constantly changing and you want your website to keep up with the latest technology and laws.

social media

These days, websites are often not enough, I can offer social media management options, and full integration into your website.
